Human Anatomy Presentation (Offline Submission)

How to run:
1) Extract the zip.
2) Open index.html in Chrome/Edge.

Controls:
- Use the top tabs to switch organs.
- Use Next/Previous or Left/Right arrow keys to navigate slides.
- Click labelled parts on the SVG to see a short popup description.
- Click any "Reference Images" box to open the image in a modal (offline).
- Fullscreen button is in the top-right.

Folder structure (do not change):
- index.html
- assets/images/ (all reference images)
